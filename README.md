## ğŸ“ System Workflow

1. Admin creates a shipment and assigns it to an agent.
2. Shipment is assigned a **unique tracking ID**.
3. User enters the tracking ID to view live status.
4. Delivery agent updates location and status at every stage.
5. On delivery, agent uploads Proof of Delivery.
6. User receives a delivery confirmation notification.
